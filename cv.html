项目经历

健身房数据库构建

在构建健身房数据库项目中，项目任务拆分为需求分析、概念设计、逻辑结构设计和数据库系统实现。过程参考真实的健身房运作，以 SQL Server 为平台，流程图、ER图、数据字典为工具。我负责 ER 图设计、数据字典的构建及部分代码的编写。构建了能体现会员、课程、教练、器材与各种工作人员的关系并实现信息操作的数据库。

泰坦尼克号乘客生存预测

在泰坦尼克号乘客生存预测项目中，面对利用机器学习分析乘客生存情况的任务，我独自编写函数读取、清洗数据并整理为 Passenger 类列表；实现 KNN 和逻辑回归模型及相关验证函数，计算指标评估性能；编写函数绘制 ROC 曲线辅助调优。最终成功提升预测准确性，积累了从数据处理到模型优化全流程的独立项目经验。

商标目标检测

在商标的目标检测项目中，为了找到效果最好的检测方法，我首先使用了 YOLO 的不同模型对比了效果，调用通义千问大模型批量检测，观察到大模型的分类效果较好，但检测结果低于模型，抽样发现在评估中大模型结果与人工标注习惯有偏差，因此 MAP 值较低。
